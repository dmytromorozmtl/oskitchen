# Footer Builder (roadmap)

## Сейчас

- `StorefrontFooter.blocksJson` в схеме; связь с `StorefrontPublicFooter` требует аудита при внедрении.

## Цель

- Редактор колонок, соцссылок, юридических ссылок, копирайта.
- Layout пресеты: simple / multi-column / minimal.

## Безопасность

- Только HTTPS для внешних ссылок; без `<script>` в пользовательском HTML.
