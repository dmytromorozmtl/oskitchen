# Implementation call script

1. Confirm business model, channels, weekly order volume, and launch target.
2. Ask for current source systems: spreadsheets, WooCommerce, Shopify, Square, Google Forms, manual lists.
3. Collect sample CSVs before committing to timeline.
4. Review menu cadence, fulfillment windows, cutoff rules, pickup/delivery split.
5. Confirm staff roles and training availability.
6. Define first production day scope: number of orders, labels, packing, route needs.
7. Agree on launch blockers and support escalation path.

Close with package recommendation and next artifact: data migration checklist.
