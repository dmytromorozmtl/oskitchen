# Menu Center admin guide

1. **Choose your operating story** — open **New menu** (`/dashboard/menus/new`), pick a strategy, confirm dates, create.  
2. **Attach items** — go to **Menu items** (`/dashboard/products`); products remain tied to a menu.  
3. **Set the live checkout menu** — toggle **Checkout menu** on exactly one active row in the Menu Center list.  
4. **Templates** — `/dashboard/menus/templates` applies dated scaffolds with category hints.  
5. **Detail** — `/dashboard/menus/[menuId]` shows overview + roadmap for future tabs (items, storefront, production).

Compliance: alcohol, nutrition, and allergen claims remain **operator responsibility** — see strategy warnings (e.g. `DRINKS_MENU`).
