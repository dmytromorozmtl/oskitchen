# Sales channels — business mode recommendations

Recommendations are implemented in `channelsRecommendedForBusinessType` inside `lib/channels/channel-registry.ts`.

| Mode | Intent |
|------|--------|
| Restaurant | Storefront + manual + Woo/Shopify + marketplace placeholders + POS placeholders |
| Café | Pickup channel card + Square/Shopify/Woo + manual |
| Bar | Event inquiries + POS placeholders + storefront |
| Bakery | Preorder card + Woo/Shopify + CSV + manual |
| Catering | Catering inquiries + storefront + CSV + manual |
| Meal prep | Storefront + Woo/Shopify + CSV + Uber Direct |
| Ghost / cloud / multi-brand | Marketplaces + storefront + ecommerce |

Copy must never promise alcohol delivery compliance beyond operator responsibility.
