# Ideal customer profile (ICP)

## Primary

- **Weekly meal prep brands** doing 100–5k orders/month across Shopify/Woo with internal kitchens.
- **Catering operators** juggling deposits + production waves.

## Secondary

- Multi-location ghost kitchens coordinating shared supplier inventory.

## Non-target (for now)

- Single-location quick-service restaurants needing in-store POS replacement.
- Brands without owned fulfillment (pure marketplace dropship).

## Buying triggers

- Spreadsheet breakage at menu versioning.
- Mis-picks / refund spikes post scaling.
- Need for branded packing artifacts.
