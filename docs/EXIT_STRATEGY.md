# Exit strategy

Potential acquirers:
- POS companies.
- Restaurant software platforms.
- Meal prep platforms.
- Shopify/WooCommerce app ecosystems.
- Delivery/logistics platforms.
- Vertical SaaS rollups.
- Agencies with restaurant clients.

Metrics that improve valuation:
- Retention and expansion revenue.
- Activated accounts.
- Order volume processed.
- Integrations connected.
- Low support burden.
- Gross margin and MRR quality.

Value-enhancing assets:
- Clean tenant architecture.
- Audit logs.
- API platform.
- Integration docs.
- Implementation playbooks.
- Low customer concentration.
