# CSV import templates

Route: `/dashboard/import-center/templates`

Templates:
- `products.csv`
- `customers.csv`
- `orders.csv`
- `ingredients.csv`
- `recipes.csv`
- `staff.csv`

Guidance:
- Keep one header row.
- Use ISO dates (`YYYY-MM-DD`).
- Use numeric prices without currency symbols when possible.
- Keep SKUs stable across channels.
- Convert XLSX tabs to separate CSV files before upload.
