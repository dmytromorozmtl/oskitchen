# Demo call script (discovery + demo)

## Opening (2 min)

“Thanks for joining — goal is to see if centralizing incoming orders before production would save you time. I’ll ask two questions, show the product for ~10 minutes, then discuss fit.”

## Discovery questions

1. What channels produce orders today?
2. Where do failures happen (missed orders, wrong qty, wrong bag)?
3. Who runs production vs packing vs integrations?
4. What tools do you use now (POS, spreadsheets, Shopify apps)?
5. Timeline if we solved this tomorrow?

## Transition

“Based on that, here’s how OS Kitchen routes orders → production → packing.” *(Follow `docs/DEMO_SCRIPT.md`)*

## Pricing bridge

“Starter is for proving workflow; Pro adds omnichannel volume; Team unlocks Uber modules + advanced ops.”

## Close

“Does a pilot workspace this week feel reasonable? I’ll send invite + webhook checklist.”
