- **Phone / counter:** `/dashboard/orders/quick` + manual `Order` creation (no `ExternalOrder` unless explicitly modeled)
- **Email orders:** Placeholder channel — operational guidance to paste into manual order notes; optional AI only when `OPENAI_API_KEY` exists (never implied as compliant parsing)

All manual paths must continue to feed **Order hub** without bypassing production controls.
