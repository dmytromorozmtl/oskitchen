# Mobile, tablet, device modes

## Targets

- POS tablet layout — large tap targets; minimal dense admin chrome on cashier surfaces.  
- KDS — readable at arm’s length; live/polling indicator.  
- Packing scanner — focus order + SKU verification; accessible error tones/copy.  
- Driver mode — least privilege; hide sensitive customer PII beyond stop needs.  
- Manager mobile Today — role-aware quick actions.

## Degraded experience

- Offline or slow network — explicit banners; no silent wrong state.  
- Optional providers missing — setup CTA, not crash.
