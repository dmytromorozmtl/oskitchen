# Product roadmap (rolling)

## Near term (0–6 weeks)

- Order hub: import external orders → internal orders + line items.
- Product mapper UI leveraging `ExternalProduct.mappedProductId`.
- Webhook replay / retry controls.
- Menu channel publish screens (`MenuChannelPublish`).

## Mid term (6–12 weeks)

- Production batches & ingredient demand (inventory lite).
- Packing labels & QR verification.
- CRM profiles consolidated across channels.
- Analytics charts (Recharts) fed by materialized summaries.

## Longer term

- Multi-location tenants & staff RBAC hardening.
- Enterprise SSO + audit trails.
- Public operator API for custom integrations.

Roadmap priority follows: **Order hub → integrations reliability → production → packing → analytics**.
