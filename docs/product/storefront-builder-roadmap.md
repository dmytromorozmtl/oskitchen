# Storefront Builder Roadmap

## Current State

Existing surface:

- `/dashboard/storefront/builder`

OS Kitchen already owns a major competitive layer by having a native storefront
builder instead of outsourcing storefront ownership to Shopify or WooCommerce.

## Next Enhancements

1. drag-and-drop section ordering
2. live preview
3. mobile preview
4. section visibility toggles
5. theme preview

## Why This Matters

Storefront ownership is one of OS Kitchen's strongest differentiators:

- revenue control
- operator lock-in through owned commerce surface
- lower plugin sprawl
- tighter linkage with production and order routing

## Recommended Rollout

- keep current builder usable
- add drag-and-drop and preview progressively
- avoid claiming "full visual no-code builder" until the interactions are truly production-polished
