# Competitor Gap Roadmap

## Strategic Position

OS Kitchen is strongest where competitors split the operator workflow across
multiple tools:

- storefront
- POS
- production
- packing
- routes

The roadmap should preserve that end-to-end advantage while closing the most
commercially important gaps.

## Recommended Sequence

1. Close `AI scheduling + HACCP` first.
2. Then strengthen `invoice-to-inventory + waste + route optimizer`.
3. Then expand `QR ordering + labor compliance`.
4. Treat `GL/native mobile/kiosk` as important, but not first before the above.
