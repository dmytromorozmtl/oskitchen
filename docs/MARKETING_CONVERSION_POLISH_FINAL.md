## Principles

2. **CTA consistency** — Primary: **Book demo** / **Start**; Secondary: **View pricing** / **Explore demo**.  
3. **Above-the-fold** — State the operator outcome first (“Run production, packing, and routes in one place”).  
4. **SEO** — ensure `metadata` on top marketing routes stays unique per page (incremental fixes).
