## Forbidden competitor claims (Era 17)

Do **not** sell or publish these without new proof artifacts:

- Toast or Square **hardware parity**, offline POS, or terminal ecosystem equivalence
- Production **SSO**, **SOC2 Type II**, or **SCIM**
- **Unified inventory** or **unified rewards** across POS and storefront
- **Rush-hour KDS** certification or Toast expo parity
- **Live DoorDash/Uber/Grubhub** marketplace operations
- **Public API enterprise SLA**
- Any row in this matrix marked as a gap **closed** without pilot or staging evidence

Validation: `npm run smoke:competitor-feature-gap-matrix`; `MARKETING_CLAIMS_STRICT=1 npm run verify-claims`.
