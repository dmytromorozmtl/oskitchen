# Costing reports

Current reporting surfaces:

- **Command center** — latest run KPIs + item table.  
- **Menu tab** — aggregated margin / profit rollups (per-unit, not mix-weighted).  
- **Workspace reports** — `/dashboard/reports` (existing).  
- **Executive** — `/dashboard/executive` (existing).  
- **Export** — `/dashboard/import-export` (costing CSV presets to be extended in import center).

Dedicated “profitability PDF” exports are **not** yet shipped; schema and engine are ready for CSV/PDF generators.
