# Enterprise security roadmap

Future enterprise controls:
- SSO/SAML.
- SCIM provisioning.
- SOC 2 readiness path.
- Expanded audit logging.
- Fine-grained RBAC.
- Encryption review.
- Data retention/deletion automation.
- Backups and restore testing.
- Incident response plan.
- Vendor management.
- Privacy policy review.

Current state does not claim formal compliance certification.
