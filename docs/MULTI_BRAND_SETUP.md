# Multi-brand setup workflow

**Route:** `/dashboard/brands/multi-brand-setup`

Guided narrative covering:

1. Brand inventory  
2. Location topology (shared vs distributed kitchens)  
3. Menu item sharing strategy  
4. Inventory model  
5. Storefront topology (per-brand domains vs multi-brand hub)  
6. Sales channel mapping  
7. Production board strategy (shared stations + badges vs dedicated boards)  
8. Reporting prerequisites (`brand_id` on orders)

Outputs are **recommendations** plus deep links to templates and assignment tools — no automatic structural changes.

See also `docs/BRAND_STOREFRONTS.md` for domain patterns.
