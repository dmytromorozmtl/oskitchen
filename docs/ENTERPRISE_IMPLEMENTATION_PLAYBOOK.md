# Enterprise implementation playbook

Enterprise implementation expands the existing checklist into:
- stakeholder list
- departments
- multiple locations
- go-live waves
- data migration phases
- training cohorts
- launch calendar
- risk register
- sign-off checklist

Recommended sequence:
1. Executive discovery.
2. Data audit and imports.
3. Integration credential readiness.
4. Pilot location/brand.
5. Training cohort.
6. Test run.
7. Wave launch.
8. Customer success handoff.

Do not mark complete until risks are mitigated and sign-offs are recorded.
