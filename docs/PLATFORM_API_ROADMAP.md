# Platform API roadmap

Vision: make KitchenOS a developer platform for food operations after checkout.

Roadmap:
- Public API v1 hardening.
- API keys with rate limits and scoped permissions.
- Webhook subscriptions with signed payloads.
- Partner integration registry.
- Developer docs.
- Embedded widgets for storefront/order tracking.
- Integration SDK for agencies and partners.
- Future app marketplace.

Do not promise general availability until rate limiting, audit logs, scopes, and tenant tests are complete.
