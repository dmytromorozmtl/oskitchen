# Competitive landscape (high level)

| Alternative | Limitation vs OS Kitchen |
|-------------|-------------------------|
| Spreadsheets | No real-time webhook ingestion, weak audit trail |
| Generic POS | Optimized for front-of-house, not batch production |
| Shopify / Woo alone | No operator cockpit for kitchen timing + packing |
| Catering suites | Narrow workflows; weak commerce connector breadth |
| Delivery marketplaces | Own demand; OS Kitchen sits beside them |
| Restaurant POS | Table service assumptions; poor preorder orchestration |

OS Kitchen wins when **orders originate everywhere** but **fulfillment must be one coherent system**.
