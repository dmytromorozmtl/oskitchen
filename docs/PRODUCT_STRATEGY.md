# Product strategy

Wedge: help food businesses migrate from spreadsheets/manual workflows into a kitchen operations dashboard.

Expansion:
1. Self-serve storefront/manual orders.
2. WooCommerce/Shopify integration.
3. Implementation packages.
4. Team production/packing/routing.
5. Partner agency dashboard.
6. Enterprise API and multi-location workspace model.

Risks:
- Overpromising integrations.
- Underestimating support load.
- Legal/compliance review lag.
- SMB churn after busy seasons.
