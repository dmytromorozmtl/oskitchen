# Nutrition & allergen labels

## What it does

- **Command center** (`/dashboard/nutrition-labels`) — KPI cards, needs-review queue, starter templates, packing + storefront policy forms.
- **Item workspace** (`/dashboard/nutrition-labels/items/[productId]`) — nutrition draft, structured allergens, ingredient declaration, verification buttons, print jobs, audit log.
- **Print queue** (`/dashboard/nutrition-labels/print-queue`).
- **Import stub** (`/dashboard/nutrition-labels/import`) — CSV header template.
- **Reports** (`/dashboard/nutrition-labels/reports`) — light rollups.

KitchenOS **does not certify regulatory compliance** — see `NUTRITION_AND_ALLERGEN_SAFETY_PRINCIPLES.md`.
