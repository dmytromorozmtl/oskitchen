# WooCommerce listing copy draft

Extension name: OS Kitchen for WooCommerce

Tagline: Send WooCommerce food orders into kitchen production and packing workflows.

Short description: OS Kitchen helps preorder food businesses fulfill WooCommerce orders with production boards, labels, and reports.

Long description: Use WooCommerce REST API keys and webhooks to bring order/catalog data into OS Kitchen. Resolve product mappings, stage imported orders, and operate production/packing from one dashboard. No official extension approval is claimed until reviewed.

- Order hub
- Product mapping
