# Catering quotes

## What it does

Creates **`CateringQuote`** + optional starter **`CateringQuoteItem`**, exposes a signed-safe **`/quote/[token]`** guest page, and lets operators mark quotes as sent while composing richer line editors internally.

## Setup

Dashboard → **Catering** → fill guest meta + optional first line → share the generated link.

## Limitations

- Email delivery & e-sign acceptance remain placeholders — operators coordinate offline.

## Future improvements

- Multi-line editor, deposits via Stripe, PDF attachments.
